What do I use?

I use Java, due to its Strongly typed and efficient nature. Its OOP based programming and extensive libraries are my reason for coding various things in it.

I use Python for its huge library count. While this language may be slow, for small programs it is extremely convenient. If you need to do something, there's a library for it.

For creating websites, HTML/CSS is my go-to. They all suck, but at least it's better than the alternatives. The CSS frameworks are nice as well, using one for this website.

JavaScript is something I only use when I really need to. I try to avoid it as much as possible since it does not align with my programming ideals. Nevertheless, I can still program in it.

Environments

I use intelliJ (with the Python plugin) for both my Java and Python development. The IDE offers a simplistic UI, which in my opinion is great because then you won't have to look for an option in huge menu with 20+ options each. The IDE also gives nice suggestions on how to optimize something in your code. And this is just the Community Edition.

I use Webstorm for all of my web-development projects. I have made some very silly things in this, like a tic-tac-toe game and connect-4. Seriously though, this is amazing. JetBrains Intellisense is really a godsend that speeds up every project. Using it for this very website!
