Here are some of my projects I've worked on.

I made a Tic-Tac-Toe game using HTML, CSS and JavaScript. This game features clicking on empty spots and the ability to play against each other. This, combined with the automatic reset makes for a fun experience.

I made Master-Mind in HTML, CSS and JavaScript. This game features selecting colors. When all the spots are filled, the game evaluates your input and compares it to the random code that has been generated. Black means correct, white means present and grey means nonexistent.

I made the Pink Goose website, fully functional with links to podcasts, blogs, and more. It features buttons and animations with loads of features. This grade-A website about cryptocurrencies and such has a great reach.

I made a Pong game using Java, with libGDX. You may only be able to play this masterpiece on PC. Using libGDX's built-in html builder I was able to embed this here. Click into it to start playing! W/S for left, and UP/DOWN for right.
